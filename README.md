Para compilar os códigos, é necessário ter instalado o compilador para C++.

Após estar instalado, basta ir no diretório "Auto-Comp" e executar o arquivo com base em seu sistema operacional:

Linux -> compile.sh
Windows (moderno) -> compile.ps1

Caso esteja usando o windows, e o compile.ps1 não opere em seu sistema, execute o arquivo "compile.bat".
